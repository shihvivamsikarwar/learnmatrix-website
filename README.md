📘 LearnMatrix - Smart Learning for Smart Career
🌐 Project Overview

LearnMatrix is an educational platform designed to provide students with structured study material, interactive training modules, and career-focused learning.
This repository contains the website codebase for the LearnMatrix platform.

🚀 Features (so far)

🏠 Homepage (Coming Soon)

📩 Contact Form (Completed)

🎨 Tailwind CSS for modern UI

⚡ Responsive design for mobile & desktop

🔗 GitHub integrated project tracking

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript, React (planned)
